# Student Life Planner Dashboard

Student Life Planner Dashboard is a responsive web application designed for college students who want to organize their academic work in one clean and simple place. The app helps users manage tasks, track deadlines, save short study notes, and view their progress through a clear dashboard layout.

## Project Overview

The purpose of this project is to create a complete interactive web application using **HTML, CSS, and Vanilla JavaScript**. This planner is made for busy students who need a practical tool to manage coursework on both desktop and mobile devices.


## Features

- Add a new task
- Edit a task
- Delete a task
- Mark tasks as completed or pending
- Add a due date for each task
- Add short study notes
- Filter tasks by all, completed, or pending
- Search tasks and notes
- Show total, completed, and pending task counts
- Show upcoming deadlines
- Show recent notes
- Focus mode toggle
- Section switching for Tasks, Deadlines, Notes, and Settings
- Save all task data using localStorage
- Responsive design for mobile, tablet, and desktop

## Technologies Used

- HTML
- CSS
- Vanilla JavaScript
- localStorage

## File Structure

- `index.html` – main structure of the app
- `style.css` – full styling, layout, transitions, and responsive design
- `script.js` – app logic, DOM manipulation, filters, localStorage, and section switching

## How It Works

The user can create tasks with a title, due date, and short note. Each task is displayed in the dashboard as a card. The user can then mark it as completed, edit it, or delete it. The app automatically updates the statistics, deadlines section, and notes section.

All task data is stored in localStorage, which means the data stays saved even after refreshing the page.

## Responsive Design

The application is fully responsive and adapts to different screen sizes:

- **Mobile:** bottom navigation and stacked layout
- **Tablet:** wider content layout with better spacing
- **Desktop:** sidebar navigation, dashboard grid, and right-side information panels
